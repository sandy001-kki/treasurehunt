# 🎉 Treasure Hunt Winner Page

A university treasure hunt winner celebration page — triggered when a participant scans the hidden QR code.

## Features

- 🎆 Full-screen canvas fireworks animation with Web Audio API sounds
- ⏳ 20-second celebration before the registration form appears
- 🏆 Glassmorphism winner registration form (Name, Year, Register Number, Section, Department)
- 📧 Form submissions delivered via Formspree to the organiser's email
- 🎊 Victory confirmed screen with confetti burst and fanfare
- 📱 Fully mobile-responsive — optimised for phones scanning QR codes
- ⚡ Zero dependencies — pure HTML, CSS, and vanilla JavaScript
